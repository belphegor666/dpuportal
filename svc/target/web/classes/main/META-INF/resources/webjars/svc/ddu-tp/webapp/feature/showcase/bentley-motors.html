Bentley Motors

The challenge

From the Bentley CIO - How can we improve our business using public data?

The answer

Use house price data and social media analysis to profile prospects based on affordability and enthusiasm for Bentley.

The WOW factor

We proposed a new concept - Bentley Live. Using the Atos Industrial Data Analytics proposition, we can create a dashboard to beam telematics from a bentley back to a pit crew in the Bentley Factory. This feature enables Bentley Owners to keep their beloved cars in perfect condition.
