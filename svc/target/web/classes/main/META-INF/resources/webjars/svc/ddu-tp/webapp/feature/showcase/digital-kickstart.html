Digital in a Day Kickstart

The challenge

How can we get a functioning prototype (including common features) live on day 1 of a project?

The answer

The Atos Kickstart Template system.

Key features

- Secure registration and log-on

- Role-based access to menus and content

- Email generation using parameterised templates

- Dashboards and charts including drill-down

- Data import and export to spreadsheet

- Opensource technology including PAMM & Bootstrap
